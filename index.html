Content-based recommendation system

System rekomendacji oparty na treści

The movie recommendation system, which allows a user to discover movies that could pique their interest. A user-friendly interface written with HTML, CSS, and JavaScript, facilities the process of finding movies that match the user's interest and making it a better experience. Flask-based API backend coded in Python is responsible for creating the connection between the Frontend and the recommendation system. The technique that I implemented for the recommendation system is content-based filtering.

System rekomendacji, który pomaga użytkownikowi odkrywać filmy dopasowane do jego zainteresowań. Intuicyjny interfejs stworzony w HTML, CSS i JavaScript sprawia, że proces wyszukiwania nowych tytułów jest wygodny i przyjemny. Backend oparto na bibliotece Flask, która umożliwia sprawną komunikację między frontendem a systemem rekomendacji. W implementacji zastosowano podejście content-based filtering, dzięki czemu rekomendacje są dostosowane do preferencji użytkownika.

AI job preperation using Pytorch library

AI chatbot z użyciem bilbioteki PyTorch

A chatbot that allows users to prepare for jobs in the AI or Machine Learning related field. a custom JSON file containing questions and responses that a job candidate might find beneficial and easy to use. The JSON file was processed using Natural Language Processing(NLP) techniques, such as tokenizing sentences with the NLTK library, lowercasing words, and applying lemmatization to reduce words based on the context. After going through this process, all the questions were reconstructed into bags of words.model using the PyTorch library. The model consists of linear layers and was trained to recognize correct tags for questions. A user interface was written in HTML, CSS, and JavaScript, while the backend API uses Flask.

Chatbot wspierający przygotowanie do pracy w sektorze AI. Zawiera własnoręcznie stworzony plik JSON z pytaniami i odpowiedziami, które mogą pojawić się podczas rozmowy kwalifikacyjnej. Plik ten został przetworzony za pomocą technik Natural Language Processing (NLP), takich jak tokenizacja zdań przy użyciu biblioteki NLTK, redukcja słów oraz lematyzacja, aby dostosować wyrazy do ich kontekstu. Po tej obróbce wszystkie pytania zostały przekształcone w bag of words. Model został zaimplementowany z wykorzystaniem biblioteki PyTorch. Składa się z warstw liniowych i został wytrenowany tak, aby przypisywać odpowiednie tagi do pytań użytkownika. Interfejs aplikacji został stworzony w HTML, CSS i JavaScript, natomiast backend działa w oparciu o Flask, zapewniając płynną komunikację między użytkownikiem a systemem.

Image Classification with CNN architecture

Klaysfikacja obrazów z użyciem architektury CNN

An image classification model was trained to recognize fish species from a dataset. A confusion matrix, a classification report, and charts visualizing loss and accuracy across multiple epochs were used to evaluate the model's performance and effectiveness. The model was created using Convolutional Neural Network (CNN) architecture and trained with a step learning rate and the Adam optimizer. Categorical Cross-Entropy was utilised as the loss function.

Model został wytrenowany, aby rozpoznawać ryby na podstawie zbioru danych. Macierz pomyłek, raport klasyfikacji oraz wykresy przedstawiające zmiany dokładności i strat w kolejnych epokach zostały wykorzystane do oceny efektywności i wydajności modelu. Model został stworzony przy użyciu architektury Convolutional Neural Network (CNN) oraz optymalizatora Adam. Jako funkcję strat zastosowano Categorical Cross-Entropy.
